================================================================================
TROUBLESHOOTING
================================================================================

Q: Cannot access admin dashboard
A: Make sure you're logged in as admin
   Check that admin account was created successfully

Q: User cannot login after approval
A: Verify account is marked as verified
   Try clearing browser cookies
   Check that user is using correct credentials

Q: Cannot create admin account
A: Run db_setup.php first
   Check that no admin account already exists

Q: Pending verifications not showing
A: Refresh the page
   Check database connection
   Verify users are actually pending
