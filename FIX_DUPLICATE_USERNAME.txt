FIX: Duplicate Username Error
================================================================================

PROBLEM:
Fatal error: Uncaught mysqli_sql_exception: Duplicate entry 'Jordan' 
for key 'username' in C:\xampp\htdocs\OLQS\create_admin.php:40

CAUSE:
The username you tried to use (e.g., "Jordan") already exists in the database.
Usernames must be unique in the system.

================================================================================
SOLUTION: USE A DIFFERENT USERNAME
================================================================================

The username "Jordan" is already taken by another user account.

CHOOSE A DIFFERENT USERNAME:
Examples:
✓ admin
✓ administrator
✓ admin_user
✓ system_admin
✓ superadmin
✓ root_admin
✓ manager
✓ admin2025

STEPS TO FIX:
1. Go to: http://localhost/OLQS/create_admin.php
2. Fill in the form with:
   - Full Name: Your Name
   - Username: (use a DIFFERENT username, not "Jordan")
   - Email: admin@example.com
   - Password: (min 6 characters)
   - Confirm Password: (repeat)
3. Click "Create Admin Account"
4. Success! ✓
