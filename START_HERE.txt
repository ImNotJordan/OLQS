WHAT IS OLQS?
================================================================================

OLQS is a web-based learning management system that allows:

TEACHERS:
  • Upload lessons (PDF, videos, presentations, images)
  • Create quizzes with automatic scoring
  • Track student performance
  • View detailed analytics

STUDENTS:
  • Access lessons
  • Take quizzes
  • Get instant feedback
  • Track progress

================================================================================
QUICK START (3 STEPS)
================================================================================

STEP 1: Initialize Database
────────────────────────────
1. Open your browser
2. Go to: http://localhost/OLQS/db_setup.php
3. Wait for "Database setup completed successfully!"

STEP 2: Create Your Account
────────────────────────────
1. Go to: http://localhost/OLQS/
2. Click "Register"
3. Choose: Teacher or Student
4. Fill in your information
5. Click "Register"

STEP 3: Start Using
────────────────────────────
1. Login with your credentials
2. Explore your dashboard
3. Start creating or learning!

================================================================================
FIRST TIME USERS
================================================================================

IF YOU'RE A TEACHER:
  1. Login to your teacher account
  2. Go to "Lessons" → Click "Upload Lesson"
  3. Upload a PDF, video, or presentation
  4. Go to "Quizzes" → Click "Create Quiz"
  5. Add questions and options
  6. Students can now take your quiz!

IF YOU'RE A STUDENT:
  1. Login to your student account
  2. Go to "Lessons" to view available lessons
  3. Go to "Quizzes" to take available quizzes
  4. Submit quiz to see instant results
  5. Check "Dashboard" for your progress

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Connection failed" error
Solution: Make sure MySQL is running in XAMPP

Problem: Cannot upload files
Solution: Check file size (max 50MB) and file type

Problem: Cannot login
Solution: Clear browser cookies and try again

Problem: Database error
Solution: Run db_setup.php again

For more help, see SETUP_INSTRUCTIONS.txt
