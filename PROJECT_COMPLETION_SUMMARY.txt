PROJECT OVERVIEW
================================================================================

A comprehensive, production-ready Online Learning and Quiz System built with
PHP, MySQL, HTML, CSS, and JavaScript. The system enables teachers to create
and manage lessons and quizzes while allowing students to learn and assess
their knowledge through an intuitive web interface.

Technology Stack:
- Backend: PHP 7.4+
- Database: MySQL 5.7+
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Server: Apache/XAMPP

================================================================================

✅ Separate accounts for teachers and students
