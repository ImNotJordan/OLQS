================================================================================
REQUEST/RESPONSE CYCLE
================================================================================

1. USER REQUEST
   Browser → HTTP Request → Web Server

2. ROUTING
   Web Server → Route to PHP File

3. AUTHENTICATION
   Check Session → Verify User Role

4. PROCESSING
   Execute Business Logic → Query Database

5. RESPONSE
   Generate HTML/JSON → Send to Browser

6. RENDERING
   Browser → Parse HTML/CSS/JS → Display
