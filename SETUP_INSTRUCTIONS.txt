================================================================================
ONLINE LEARNING & QUIZ SYSTEM (OLQS) - SETUP INSTRUCTIONS
================================================================================

QUICK START GUIDE:

1. INITIALIZE DATABASE
   - Open your browser and go to: http://localhost/OLQS/db_setup.php
   - Wait for the message "Database setup completed successfully!"
   - This will create all necessary tables and database

2. ACCESS THE APPLICATION
   - Go to: http://localhost/OLQS/
   - Click "Register" to create a new account
   - Choose account type: Teacher or Student
   - Complete registration and login

3. TEST THE SYSTEM
   
   For Teachers:
   - Login with teacher account
   - Go to "Lessons" → "Upload Lesson" to upload files
   - Go to "Quizzes" → "Create Quiz" to create a quiz
   - Add questions with multiple choice, true/false, or short answer types
   - Go to "Analytics" to view student performance
   
   For Students:
   - Login with student account
   - Go to "Lessons" to view available lessons
   - Go to "Quizzes" to take available quizzes
   - View results immediately after submission
   - Check "Dashboard" for progress statistics

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Connection failed" error
Solution: 
- Ensure MySQL server is running
- Check database credentials in config.php
- Verify XAMPP services are started

Problem: Cannot upload files
Solution:
- Check uploads/ directory permissions (should be 755)
- Verify file size is under 50MB
- Confirm file type is supported

Problem: Login not working
Solution:
- Clear browser cookies
- Verify username and password are correct
- Check user account was created successfully

Problem: Database tables not created
Solution:
- Run db_setup.php again
- Check MySQL error logs
- Ensure database user has CREATE TABLE permissions
